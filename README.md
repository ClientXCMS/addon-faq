# faq

## Description

This extension provides additional functionality for ClientXCMS.

## Installation

1. Clone this repository into your ClientXCMS installation:
   ```bash
   cd /path/to/clientxcms
   git clone git@github.com:ClientXCMS/addon-faq.git
   ```

2. Copy the extension to the appropriate directory:
   ```bash
   cp -r addons-faq/addons/faq ./addons/faq
   ```

3. Run composer install if needed:
   ```bash
   cd addons/faq
   composer install
   ```
